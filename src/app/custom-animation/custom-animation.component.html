Custom animations

Ngx-zero-dialog comes with simple pre-built animation, take a look:

This demo page uses customized animation which plays with transform property. .ngx-zero-dialog-hidden is a starting hidden state of dialog, and .ngx-zero-dialog-visible is a final state of dialog. When enableAnimations: false is provided in config - only .ngx-zero-dialog-visible will be applied to dialog element and dialog won't play animation.
